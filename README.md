#  StockSense AI – Data-Driven Stock Assistant

##  Overview

StockSense AI is a domain-focused chatbot designed to help users analyze Indian stocks, understand market trends, and extract structured insights.

Instead of a generic chat interface, the product is designed as a decision-support tool with guided actions and structured outputs.

---

##  Why This Project

Most AI chatbots are generic and require users to figure out what to ask.

I wanted to solve this by:

* Providing predefined actions (analyze, compare, trends)
* Structuring outputs into insights and risks
* Designing a UI that feels like a real fintech product

---

##  Key Features

*  Stock Analysis (TCS, INFY, etc.)
*  Structured Insights (Key Insights + Risk Factors)
*  Sentiment Indicators (Bullish / Bearish / Neutral)
*  Smart Suggestions (guided prompts)
*  Decision-focused UI (not a chat wrapper)
*  Fully responsive design

---

##  Frontend Thinking

* No empty chat screen → guided entry points
* Structured responses instead of raw text
* Visual hierarchy for quick scanning
* Domain-specific UI (fintech feel)
* Micro-interactions and responsive layout

---

##  Tech Stack

* React + Vite
* Tailwind CSS
* Node.js (API)
* OpenRouter for Free API
